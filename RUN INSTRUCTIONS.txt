1- Make sure required python packages installed. (use "pip install -r requirements.txt" command)
2- Locate the Scraper\scrape_tweets.py
3- Run scrape_tweets.py with python scrape_tweets.py
4- After running the scraper successfuly, CSV file will be created in the same directory
5- Run the web server with the command python manage.py runserver
6- Upload the CSV file to the web server
7- After uploading you can see the tweets by navigating Display Tweets
